# pyvz-
根据植物大战僵尸脚本框架，以及前人键控脚本的改进，实测可挂机

# 脚本如何运行
把该脚本连带着pvz文件夹（下载链接在下面）一起放入pvz的目录，右键选择在终端打开，然后输入以下指令

python 脚本名称.py

# 以下是部分你可能想问的问题的解答
Q:为什么最后要是冰消珊瑚而不是炮消珊瑚？这样阳光不会负增长吗？

A:因为使用炮消珊瑚的话，在前2f可能有效果，但是下一关可能会因未知原因并没有发出那一门炮，导致两门炮被啃
经过测试阳光并不会负增长

Q:为什么在循环的开头会出现time.sleep(2)？

A:因为上述原因（Exception: 卡片数量 4 超过卡片格数 0.），本想解决无故报错，但似乎效果不大，但好在现在解决，脚本停运只需要重新开启即可
